| **7. `res.download()`**     | Sends a file for the user to download.                                     | `res.download('report.pdf')`                   |
| **8. `res.end()`**          | Ends the response (usually used in custom server logic).                   | `res.end()`                                    |
| **9. `res.cookie()`**       | Sets a cookie in the browser (used with `cookie-parser`).                  | `res.cookie('username', 'Rahul')`              |
| **10. `res.clearCookie()`** | Removes a cookie.                                                          | `res.clearCookie('username')`                  |
